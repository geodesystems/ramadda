Importing Slack Teams

You can import and display Channels and Messages from your Slack Team in RAMADDA.

To set this up you create a RAMADDA Slack Team entry and give it an API token that you get from your Slack Team.

  1. In Slack go to the Web API page and copy the API token.

  2. Now, switch over to RAMADDA and go to where you want to have your Slack Team set up. From the File->New Entry menu choose Slack Team.
  3. Enter the authentication token you got from Slack and, optionally, any channels you want displayed. If you don't enter any channel ids then all channels will be shown.

  4. Save the Slack Team entry. It should now display the selected Slack Channels.
  5. You can now import this content and display it in various ways using RAMADDA's Wiki facility
