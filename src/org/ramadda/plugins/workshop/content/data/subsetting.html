[ht::head {Grid Subsetting}] RAMADDA supports subsetting and accessing gridded data files
  1. Navigate to the AVN grid you have loaded previously
  2. Go to the [ht::menu Data {Subset Spatially}] menu
  3. Select a stride, spatial subset (maybe over Africa) and one or more fields
You can also turn around and publish the subsetted grid back to RAMADD
  1. Repeat the above steps
  2. Under "Select a folder to publish the subset to" press Select to select a Folder to publish the grid to
[ht::foot]
