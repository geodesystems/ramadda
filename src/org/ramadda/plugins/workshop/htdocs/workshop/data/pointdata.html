Section 4.0: Point Data Services
RAMADDA supports subsetting and accessing point data files
  1. Go to the top level Data folder
  2. Go to File->New File
  3. Select the earthquake netCDF file on your local machine 
         /data/idv/point/earthquake.nc
  4. Note there is now a Data menu. Select Data->NCDump to show the NetCDF metadata.
  5. Select Plot points on a map
  6. Select CSV, KML Output. Select a format and subset over a region.
