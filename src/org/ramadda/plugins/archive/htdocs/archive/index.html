+section title="Archive" :navtop :heading Archive Exhibits RAMADDA provides support for creating digital exhibits with the Archive Exhibit and Archive Gallery entry types. An Archive Exhibit can contain any number of Archive Gallery entries. The order in which the Gallery entries are shown is determined by their "entry order". To set the entry order see the User Guide :p The Archive Gallery is a special purpose Folder that can contain both it's own children entries as well as a list of entries from elsewhere in your RAMADDA (e.g., Archive Items, etc.). See the User Guide for information on defining these linked entries. :p The Gallery has a pre-defined display that, by default, shows the parent Archive Exhibit entry as the main section. It then displays the Gallery name as a sub-section with the Gallery description and an optional thumbnail image. The children entries of the Gallery (e.g., Archive Items, Images, Videos, etc) are then shown as a gallery display showing their thumbnail images (if defined). :heading Archive Library Constructs The main Archive Library constructs are- * Archive Collection * Archive Series * Archive File * Archive Item * Archive Book * Archive Object :heading Natural Science Specimens There are a collection natural science specimen entry types, e.g, Ornithology, Insect, Geologic, etc. You can view the list of entry types here.
