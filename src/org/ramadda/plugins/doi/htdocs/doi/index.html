Creating Digital Object Identifiers (DOI)

RAMADDA supports tagging entries with a Digital Object Identifier (DOI) or Archival Resource Key) (ARK). This is a service provided by EZID and allows for

You can also search for an entry with a DOI search form.

Configuring your server

To enable the DOI plugin you must have an account with EZID you need to specify the following properties in some property file in your RAMADDA home dir, e.g. ezid.properties: 
##Account information 
ezid.username=
ezid.password=

doi.adminonly=true

doi.prefixes=

#e.g.:
doi.prefixes=doi:10.5072/FK2,ark:/99999/fk4

#What metadata profile to use as the default
ezid.profile=erc
#ezid.profile=datacite
#ezid.profile=dc
