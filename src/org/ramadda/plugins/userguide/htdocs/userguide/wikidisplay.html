11.2 Wiki Display Components
RAMADDA provides a set of information and data display components in the wiki facility. There are a number of examples available. Each of these components belongs to some group. A group can layout its children component (see below) and also supports sharing of events among components (e.g., record selection in a chart). Each of these tags support the following attributes: 
layoutHere="true"  
showMenu="true"  
showTitle="true"
title=Title to display
If layoutHere is set to false then the component is laid out by its group. Otherwise it is displayed in place.

11.2.0 The Displays

  • Group
    {{group layoutType="columns" layoutColumns="2"}}

    The layoutType can be one of:

    • table
      The layoutColumns attribute specifies how many columns to use for the table.
    • rows
      Each child display specifies the row they are placed in.
    • columns
      Each child display specifies the column they are placed in.
    • tabs
      Each child display is placed in a tab.


  • Search form
    {{display type="entrylist" width="800" height="400" orientation="vertical" showForm="true" formOpen="true" }}
    Supports an embedded and configurable search form.

    Image 1: Search Form

  • Map
    {{display type="map" width="100%" #height="400" }}
    A map that shows georeferenced point data. This can be integrated with other displays, e.g., clicking on a point will highlight that location in a line chart.

    Image 2: Map

  • Line chart
    {{display type="linechart" #width="800" #height="400" #fields="" }}
    Displays time series or indexed point data.

  • Area chart
    {{display type="areachart" #width="800" #height="400" #fields="" #isStacked="true" }}
  • Scatter Plot
    {{display type="scatterplot" #width="800" #height="400" #fields="" }}

  • Histogram
    {{display type="histogram" #width="800" #height="400" #fields="" #colors="comma separated color list" #legendPosition="none|top|right|left|bottom" #textPosition="out|in|none" #isStacked="false|true|percent|relative" #logScale="true|false" #scaleType="log|mirrorLog" #vAxisMinValue="" #vAxisMaxValue="" #minValue="" #maxValue="" }}

  • Bubble Chart
    {{display type="bubble" width="500" height="500" #fields="label|x|y|color|size" #colorTable="rainbow|grayscale|inversegrayscale|blues|blue_green_red|white_blue|blue_red" #colors="comma separated color list" #legendPosition="none|top|right|left|bottom" #hAxisFormat="none|decimal|scientific|percent|short|long" #vAxisFormat="none|decimal|scientific|percent|short|long" #hAxisTitle="" #vAxisTitle="" }}

  • Pie chart
    {{display type="piechart" #pieHole="0.5" #is3D="true" #fields="" #bins="bin count" #binMin="min" #binMax="max" #sliceVisibilityThreshold="0.01" width="500" height="250" }}

  • Calendar
    {{display type="calendar" #fields="" #cellSize="15" #missingValue="0" }}
    Displays point data based on the date of each record. The calendar entry is color coded based on the specified field.

  • Gauge
    {{display type="gauge" #fields="" #gaugeMin="" #gaugeMax="" #gaugeLabel="" }}
    This shows a single value from a series of point data.

  • Sankey
    {{display type="sankey" #fields="" }}
    Displays flow data. The point data needs to have 2 string fields (the from and to) and a numeric value field.

  • Table
    {{display type="table" height="400" #fields="" }}
    Shows tabular data. This is integrated with other charts so clicking on a row highlights the corresponding record in a chart.

  • Heatmap
    {{display type="heatmap" #height="400" #fields="" #colorTables="temperature,blues" #colorByMins="-90,0" #colorByMaxes="45,100" #showIndex="false" #showValue="true" #showBorder="true" #cellHeight="30" #textColor="white" }}
    Displays the records, one row per record, with each cell color coded based on the field value and the specified color table.

  • Correlation
    {{display type="correlation" #fields="" }}
    Displays a statistical correlation of a set of fields.

  • Text Readout
    {{display type="text" }}
    Displays the record selected in other charts.

  • Summary Stats
    {{display type="stats" loadData="false" showDefault="true" #showMin="true" #showMax="true" #showAverage="true" #showStd="true" #showPercentile="true" #showCount="true" #showTotal="true" #showPercentile="true" #showMissing="true" #showUnique="true" # #showText="true" }}
    Displays a summary statistics including average, min, max, number of missing, etc.

  • Animation
    {{display type="animation" }}
    Controls time or index based animation. Other charts will highlight the record of the current time.


  • Entry Grid
    {{display type="entrygrid" showIcon="true" showName="true" scaleWidth="true" scaleHeight="false" #xAxisAscending="true" #xAxisScale="true" #yAxisAscending="true" #yAxisScale="true" #urlTemplate="{url}|{entryid}|{resource}" #height="400" #width="100%" #entries="" }}

11.2.1 Filtering Data
You can filter the data that is shown based on string pattern matching or numeric operators. You do this with the attributes: 
patternFilterField="field id" 
filterPattern="some string"
numericFilterField="other field id" 
numericFilterValue="number" 
filterOperator=">|>=|==|<|<="
e.g.: below would only show data where the 1st matches the string "Texas" and the 3rd needs to be less that 5000. 
{{display  type="sankey" showMenu="true" height="800" 
patternFilterField="#1" filterPattern="Texas"
numericFilterField="#3" numericFilterValue="5000" filterOperator=">" 
}}
