[ht::head {Multiple Languages}] :navtop +callout-info RAMADDA has support for multiple languages -callout :heading Phrase Files Language translations are defined in a set of language pack phrase files. These files are installed as RAMADDA plugins and should be called: +pre <language id>.pack or <language id>.pack.txt -pre Where <language id> is the short identifier for the language, e.g. There are a number of pre-defined language phrase files. You can view these files on Github. :p You can view the list of available languages and their phrase translations here. :heading Language Switcher To enable the display of the page language widget in the upper left of every page set the following property in a repository.properties file or in the Admin->Settings Properties section +pre addCopy=true ramadda.languages.enabled = true -pre To show a large language switcher add the wiki text: +pre :langswitcher en,es,fr -pre e.g.: +center :langswitcher en,es,fr -center If you want to add the language switcher to every page add the above langswitcher wiki text to a Page Header property. :heading Setting the Language To set the language on a page add the following wiki text: +pre :setlang <the language ID> -pre e.g. for Spanish: +pre :setlang es -pre :heading Language Blocks The wiki text of an entry can have any number of language blocks. This lets you have long form text for different languages. +pre +lang en This is the English -lang +lang es This is the Spanish -lang -pre [ht::foot]
