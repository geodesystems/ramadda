[ht::head {Tips and Tricks}] <%childlist%> :navtop :heading File upload dialog When creating a new entry or creating a thumbnail metadata the below form is shown that supports selecting a file or drag-and-dropping a file. You can also paste an image or text into the "Or drag files here" area. [ht::screenshot images/fileupload.png {File Upload}] :heading Drag and Drop in Wiki Editor The wiki editor supports drag-and-drop of files. The file is uploaded to the RAMADDA server and a new entry is created. The user is then asked what should be embedded in the wiki text, e.g.: as an image: [ht::screenshot images/dnd.png {Drag and drop}] The wiki editor also supports pasting an image that has been copied to the clipboard (e.g., through a screen grab). A new image entry is created and the above dialog is shown. :heading Search in page The is a wiki display component, display_simplesearch, that supports searching within a page. This can be seen in the Maps Gallery [ht::screenshot images/inpage.png {Search in page}] Add this to a page with: [wiki::text { {{display_simplesearch doPageSearch=true placeholder="Search in page"}} }] This supports grid, gallery and entry tree/table displays. :heading Full-screen mode In wiki text add the below to provide a full screen mode. When mousing over the component an icon is shown in the upper right. [wiki::text { +fullscreen Anything here... -fullscreen }] :heading Draft Pages Using the wiki tags +draft/-draft will show the enclosed section as a draft. [wiki::text { .... +draft Anything here... -draft ... }] :heading Skip To skip a block of text do: [wiki::text { .... +skip Anything here... -skip ... Or :skip Any line }] [ht::foot]
