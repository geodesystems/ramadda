This is integrated with other charts so clicking on a row highlights the corresponding record in a chart. } { height="400" } { images/charts/table.png}] [displayType {Heatmap} heatmap { Displays the records, one row per record, with each cell color coded based on the field value and the specified color table. } { type="heatmap" height="400" fields="" colorTables="temperature,blues" colorByMins="-90,0" colorByMaxes="45,100" showIndex="false" showValue="true" showBorder="true" cellHeight="30" textColor="white"} { images/charts/heatmap.png }] [displayType { Correlation} correlation { Displays a statistical correlation of a set of fields. } { fields="" } { images/charts/correlation.png }] [displayType { Text Readout} text { Displays the record selected in other charts. } { } { images/charts/textreadout.png }] [displayType {Canvas} canvas { The canvas display takes a specification of one or more glyphs which is used to create a customized display for each record. } {canvasHeight="100" canvasWidth="100"} {images/charts/canvas.png} {} { The glyphs are specified by a set of attributes: 
glyph1="type:<glyph type>,..."
glyph2="type:<glyph type>,..."  
...
glyphN="type:<glyph type>,..."
Where the type can be one of label, circle, rect, image, gauge, 3dbar, axis,tile, vector. e.g.: 
glyph1="type:3dbar,pos:sw,dx:10,dy:-5,height:100,width:8,baseHeight:5,sizeBy:atmos_temp,sizeByMin:0,sizeByMax:50"
glyph2="type:label,font:6pt,pos:ne,dx:90,dy:-90,label:Temp: ${atmos_temp}"
} ] Each of these glyphs is positioned within the canvas based on the attributes for the glyph.
  • pos:n or nw or w or sw or s or se or e or ne
    This is the position on the bounding rectangle for the glyph.
  • dx,dy: These are the delta x and delta y values which are applied to the origin of the canvas where the origin is the lower left corner of the canvas. These can be either positive or negative integer values but can also be the following literal values:
    • canvasWidth,-canvasWidth: the width of the canvas
    • canvasWidth2,-canvasWidth2: half the width of the canvas
    • canvasHeight,-canvasHeight: the height of the canvas
    • canvasHeight2,-canvasHeight2: half the height of the canvas
For example, say you want to have a label that is position in the upper right of a canvas of width and height 100. To do this specify the following. The dy=-90 takes into account the approximate height of the text block. 
glyph1="type:label,pos:ne,dx:100,dy:-90,label:Some label"
