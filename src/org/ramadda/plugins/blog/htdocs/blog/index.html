RAMADDA Blogs

You can have any number of web logs in RAMADDA. Just go to any folder, press File->New Entry. In the big list select Weblog.

Configuring you weblog

The default UI is pretty simple. You can change the header and the side navigation through the Edit Entry page for the Web log entry. If you want to change the header with your own image you can upload your header image as a File Attachment from the Edit->Add Properties menu (under "Thumbnails, Page Style, etc."). Then, edit the Web log entry, Add the following wiki text in the header: 
{{image src="::<filename>.png" align="center"}}
The navigation text is also treated as Wiki text.
