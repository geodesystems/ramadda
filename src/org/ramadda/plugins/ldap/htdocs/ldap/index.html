LDAP Based User Authentication

LDAP User Authentication

You can configure RAMADDA to use an LDAP server for user authentication.
  • The LDAP plugin comes in the Core plugins. If you don't have the Core plugins installed then install the ldapplugin.jar from the SourceForge site into the RAMADDA home/plugins directory.
  • Copy the example ldap.properties file into the RAMADDA home directory.
  • Edit the settings in the property file to define the LDAP server and user configurations.
