The Database Table entry in RAMADDA is an admin-only entry that allows you to display the contents of an external database table.

Configuration

The RAMADDA server has to be configured with the database connection information. Create a .properties in the RAMADDA home directory with the following. 
#Comma separated list of one or more available databases by some unique name
table.db.databases=employees,testdb

#For each of the databases above specify the JDBC url, the user name (if needed) and password
table.db.<dbname>.url=<jdbc URL>
table.db.<dbname>.user=
table.db.<dbname>.password=

#e.g.

table.db.employees.url=jdbc:mysql://localhost:3306/employees?zeroDateTimeBehavior=convertToNull&initialTimeout=1000&connectTimeout=5000
table.db.employees.user=
table.db.employees.password=

table.db.testdb.url=jdbc:mysql://localhost:3306/testdb?zeroDateTimeBehavior=convertToNull&initialTimeout=1000&connectTimeout=5000
table.db.testdb.user=
table.db.testdb.password=
Once configured you can create a Database Table entry (under Documents). You need to specify the database and the table to display along with other display settings. Note: Select the database then save the Entry. Then go and Edit the entry and a pull-down list of the available database tables will be shown.
