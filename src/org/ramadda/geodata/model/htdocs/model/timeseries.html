This form allows you to plot a single field from a model/experiment or compare one field to itself or another. The instructions below explain each of the sections of the form.

  • Field(s)
  • Time Series
  • Make Time Series

Field(s)

  1. From the Collection, select the Model, Experiment, Ensemble Member and Variable you wish to plot
  2. Optionally, select a second data set you wish to compare with the first.
  3. Click the Select Data button to bring up the options for analyzing and plotting the data you've selected. If you make any changes in the Field(s) section, click the Update Data Selection button to refresh the options below.

Time Series

  1. If you are plotting pressure level data, select the Level you wish to use
  2. You can plot either the Average or the Anomaly over the region. Anomalies are computed relative to 1981-2010 average for the ensemble mean.
  3. Choose a starting and ending month over which to average the data.
  4. Choose a starting and ending year for the time series.
  5. Select the Region that the data will be averaged over. You can select from a predefined list of regions, or select Custom. If you choose custom, you can draw a region on the map (use Shift+drag to draw the region), or enter the North, South, West and East coordinates of the region.

Make Time Series

  1. You can select to plot a shaded area between the ensemble min and max values, or to plot each line individually.
  2. For either plot, you can select the color of the shaded area or the ensemble lines. The mean is always plotted black.
  3. For some fields, you can select the output units for the plot.
  4. You can select a mask for the data to block out either ocean or land areas.
  5. You can override the default range for the Y axis (data range).
  6. By default, a 5-year running average is plotted to smooth out the lines. You can select a longer or shorter period.

Make Time Series

  1. After you have made your selections, click the Make Time Series button to generate the time series image.
  2. An image will be displayed to the right of the data selection region. You can click on the image to get a full sized version of it. There is link below the image which allows you to download the image to your computer.
