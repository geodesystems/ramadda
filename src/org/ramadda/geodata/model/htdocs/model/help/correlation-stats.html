Time Series Comparison

  1. If you are plotting pressure level data, select the Level you wish to use
  2. For times, you can:
    • Choose a starting and ending month over which to correlate the data.
    • Choose a starting and ending year over which to correlate the data.
    • Additionally, you can select a lead/lag period for the time series. None is the default and uses the same months/years as the model data. Lead indicates that the time series leads the model data in time (i.e, is a predictor of the model data). Lag indicates that the time series lags the model data (i.e., the model predicts the state of the time series). If you select Lead or Lag, then you can select the months of the time series lead/lag period. If no months are selected, the model months are used with a 1 year lead or lag depending on which option you chose.
  3. Select the Region that the data will be averaged over. You can select from a predefined list of regions, or select Custom. If you choose custom, you can draw a region on the map (use Shift+drag to draw the region), or enter the North, South, West and East coordinates of the region.
