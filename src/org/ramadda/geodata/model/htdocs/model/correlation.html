This form allows you to plot a single field from a model/experiment or compare one field to itself or another. The instructions below explain each of the sections of the form.

  • Field(s)
  • Time Series Comparison
  • Plot Options
  • Make Plot

Field(s)

  1. From the Collection, select the Model, Experiment, Ensemble Member and Variable you wish to plot. You can select one or more ensemble members, however, if you select more than 10-15, the process will most likely time out.
  2. From the Time Series, select the Climate Index you wish to compare with the model data. A description of the timeseries can be found here.
  3. Click the Select Data button to bring up the options for comparing and plotting the data you've selected. If you make any changes in the Field(s) section, click the Update Data Selection button to refresh the options below.

Time Series Comparison

  1. The model variable and time series you selected are listed. If you are plotting pressure level data, select the Level you wish to use.
  2. Choose a starting and ending month over which to average the data.
  3. Choose a starting and ending year for the time series.
  4. Select the Region that the data will be averaged over. You can select from a predefined list of regions, or select Custom. If you choose custom, you can draw a region on the map (use Shift+drag to draw the region), or enter the North, South, West and East coordinates of the region.

Plot Options

  1. You can select to plot the correlation or comparison between the two time series.
  2. You can select a mask for the data to block out either ocean or land areas.
  3. You can plot the output as color-filled contours and optionally overlay contour lines and labels.
  4. You can override the default contour intervals for the plots by specifying a contour interval, minimum and maximum values.

Make Plot

  1. After you have made your selections, click the Make Plot button to generate the image.
  2. An image will be displayed to the right of the data selection region. You can click on the image to get a full sized version of it. There is link below the image which allows you to download the image to your computer.
  3. There are links for downloading the individual files that were used to create the image and to download all files.
