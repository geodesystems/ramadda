Installing RAMADDA

Consult the main installation guide for basic information on how to install RAMADDA. :p To configure your RAMADDA to cooperate in the TIDE data network: * Download the TIDE RAMADDA Plugin * Create your RAMADDA home directory as described in the installation guide * Create a "plugins" directory under your RAMADDA home directory * Copy the TIDE RAMADDA Plugin to this plugins directory * Install RAMADDA The TIDE plugin will: * Add a default TIDE web page template * Provide a custom initial set of content and entries (e.g, data, documents, etc) * Specifies that your RAMADDA imports the set of shared TIDE network data policies * Add your RAMADDA to the TIDE network set of registered RAMADDAs. This uses RAMADDA's Registry Service

Data Policies

The TIDE network provides a set of shared RAMADDA Data Policies. These data policies allow you to configure access control policies, shared data citations and licenses for certain areas of your RAMADDA repository. :p The below image shows how the set of RAMADDAs in the TIDE network can share these data policies. +center -center

Federated Search

The TIDE plugin specifies that your RAMADDA registers itself with the central TIDE Registry Service. This allows other RAMADDA nodes in the TIDE network to be aware of your RAMADDA and for your RAMADDA to be aware of the set of federated TIDE RAMADDA repositories. This collection of federated repositories are then available for distributed search. :p The below image shows how this federated search works within the TIDE network +center -center -section
