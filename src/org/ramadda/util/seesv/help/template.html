For example, with this data: data:simple.csv Calling: seesv: -template "PREFIX\n" "name:\${name} value:\${value}" "\nROW DELIMITER\n" "\nSUFFIX\n" test.csv Results in: 
PREFIX
name:joe value:5
ROW DELIMITER
name:jill value:2
ROW DELIMITER
name:bill value:7
SUFFIX
Note: the row delimiter is only printed between rows.

If you wanted to generated custom XML, for example, call: seesv:seesv -template "\n" "" "\n" "\n\n" test.csv Which results in: 

<mytags>
<tag name="joe" value="5"></tag>
<tag name="jill" value="2"></tag>
<tag name="bill" value="7"></tag>
</mytags>
