This assumes you have a separate file that contains a key value and latitude/longitude and the source file has a key value. For example, you might have a list of precincts, e.g.: precinct,latitude,longitude 4171107648,40.197055,-105.100351 2171207308,39.996986,-105.084938 2173307410,40.039912,-105.081393 The source file is a list of voters, e.g.: name,precinct John Doe,4171107648 Jane Doe,4171107648 Jim Smith,2171207308 To add the lat/lon: seesv:seesv -geocodejoin precinct precincts.csv precinct latitude longitude
