For example, the below command would change the value of column2 for all rows that whose column1 value matches "string": seesv:seesv -if -match column1 string -change column2 .* XXX -endif And here is a real world example of using the -if command for USGS Stream Gauge data. This checks if there is a discharge column. If there is then it calculates a volume from the discharge. seesv:seesv -if -has discharge -integrate discharge date second volume -endif Here is an example that checks if a value is NaN and if so it prints the columns seesv:seesv -if -isnan value -printcolumns 0-5 -endif Here we do the same but set fixed widths and decorate the output seesv:seesv -if -isnan value -setting xwidths "50,>15" -setting decorate true -printcolumns 0-5 -endif
