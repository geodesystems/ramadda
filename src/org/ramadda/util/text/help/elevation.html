This uses the web service at https://nationalmap.gov/epqs/pqs.php to look up the elevation for the given lat/lon. For example if you had the following data file: name,latitude,longitude City 1,40.197055,-105.100351 City 2,39.996986,-105.084938 City 3,40.039912,-105.081393 You can add the elevation with: seesv:seesv -geoname countries latitude longitude -geoname timezones latitude longitude
