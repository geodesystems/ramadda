This concatenates the input files, passing through the first file whole but skipping over the given number of lines in the subsequent files. So with files: test1.csv: name,value foo,bar test2.csv: name,value x,y test3.csv: name,value l,m Calling: seesv -append 1 test1.csv test2.csv test3.csv Results in: name,value foo,bar x,y l,m
