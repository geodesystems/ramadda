This uses one of the shapefiles provided by RAMADDA and finds the name of the feature that contains the given lat/lon. For example if you had the following data file: name,latitude,longitude City 1,40.197055,-105.100351 City 2,39.996986,-105.084938 City 3,40.039912,-105.081393 You can add the country and timezone with: seesv:seesv -geoname countries latitude longitude -geoname timezones latitude longitude
